Formula version

R provides a standard way to describe relationships among variables through formula specification.

A formula uses the tilde (~) operator to separate response variables on the left from explanatory variables on the right. We will use this in different applications

aggregate(income ~ Segment, data=seg.df, mean)
##      Segment   income
## 1  Moving up 53090.97
## 2 Suburb mix 55033.82
## 3  Travelers 62213.94
## 4  Urban hip 21681.93

aggregate(kids ~ Segment, data=seg.df, sum)
